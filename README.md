# CodeForge

> Terminal AI Coding Companion dengan Multi-Agent Plug & Play

**Created by NanoMind - 2026**

Building the future of terminal AI coding, one keystroke at a time.

## Quick Start

    export GEMINI_API_KEY="AIzaSy..."
    codeforge

## Features

- 3-pane TUI layout (Chat | Diff | Context)
- Multi-provider: Gemini 2.5 Flash (default), Claude
- Real-time streaming response
- Git auto-commit dengan conventional messages
- Slash commands: /help /cost /about /provider /model /commit /clear /quit
- Vim-style keybindings
- Single binary 11MB (Termux/Android compatible)
